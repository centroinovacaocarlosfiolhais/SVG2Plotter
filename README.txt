SVG2Plotter v2.0
================
Visual Layout Manager para Cortadoras de Vinil HPGL
Centro de Inovação Carlos Fiolhais · CDI Portugal
Desenvolvido por David Marques — Vibe Coding with Claude.ai

CONTEÚDO DO ZIP
---------------
  svg2plotter.py                  — Aplicação principal
  setup.py                        — Instalador (cria atalho Desktop + instala deps)
  SVG2Plotter_Manual_Utilizador.pdf — Manual completo
  CH341SER.exe                    — Driver USB-Serial (adiciona tu mesmo)

INSTALAÇÃO RÁPIDA
-----------------
1. Extrai todos os ficheiros para uma pasta permanente (ex: C:\SVG2Plotter\)
2. Adiciona o CH341SER.exe a esta pasta se ainda não tiveres o driver
3. Instala o driver CH341SER.exe (necessário para Windows reconhecer a porta USB)
4. Faz duplo-clique em setup.py
5. Usa o atalho SVG2Plotter no Desktop

REQUISITOS
----------
  Python 3.8+  (python.org)
  pyserial     (instalado automaticamente pelo setup.py)
